# 🛒 Sina 401 - متجر سينا الإلكتروني

![Logo](/static/css/logo.png.jpeg)

## 📋 وصف المشروع

**Sina 401** هو متجر إلكتروني متكامل لبيع ملحقات الهواتف المحمولة والإكسسوارات. يتضمن نظام إدارة منتجات، سلة تسوق، نظام طلبات، وبوابة مسوقين بالعمولة.

## ✨ المميزات الرئيسية

| الميزة | الوصف |
|--------|-------|
| 🏪 **عرض المنتجات** | شبكة منتجات أنيقة مع بحث وفلترة |
| 🛒 **سلة تسوق** | إضافة، تعديل، وحذف من السلة |
| 📦 **نظام طلبات** | تتبع الطلبات برقم الهاتف |
| 👑 **لوحة تحكم** | إدارة المنتجات والطلبات والمسوقين |
| 👥 **نظام مسوقين** | كود خاص لكل مسوق مع إحصائيات |
| 📊 **إحصائيات** | مبيعات، طلبات، مخزون في لوحة واحدة |
| 📱 **تصميم متجاوب** | يعمل على الموبايل والتابلت والكمبيوتر |
| 🖼️ **رفع صور** | ضغط تلقائي للصور لحماية السرعة |

## 🚀 طريقة التشغيل

### المتطلبات
- Python 3.9 أو أحدث
- حساب Vercel (للنشر)
- قاعدة بيانات PostgreSQL (Supabase مجاني)

### التثبيت المحلي

```bash
# 1. نسخ المشروع
git clone https://github.com/mosebh99/sina.401.git
cd sina.401

# 2. إنشاء بيئة افتراضية
python -m venv venv

# 3. تفعيل البيئة
# على Windows:
venv\Scripts\activate
# على Mac/Linux:
source venv/bin/activate

# 4. تثبيت المكتبات
pip install -r requirements.txt

# 5. تشغيل المشروع
python app.py
```

سيفتح الموقع على: `http://localhost:5000`

### النشر على Vercel

```bash
# 1. تثبيت Vercel CLI
npm install -g vercel

# 2. تسجيل الدخول
vercel login

# 3. النشر
vercel --prod
```

## 🔌 نقاط API

|---------|--------|-------|
| GET | `/api/products` | قائمة جميع المنتجات |
